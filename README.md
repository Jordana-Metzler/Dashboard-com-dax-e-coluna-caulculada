# Dashboard de Vendas, Custos, Lucro e KPIs

Este projeto apresenta um dashboard desenvolvido no **Microsoft Power BI** para analisar o desempenho comercial de uma empresa, considerando indicadores de vendas, custos de envio, lucro e margem de lucro.

O dashboard foi construído como parte de um exercício prático de análise de dados, com o objetivo de transformar perguntas de negócio em visualizações interativas e indicadores que facilitem a tomada de decisão.

## Objetivo do projeto

O objetivo principal é analisar o desempenho das vendas ao longo do tempo e identificar como fatores como modo de envio, mercado e categoria de produto influenciam os resultados da empresa.

O dashboard permite responder às seguintes perguntas de negócio:

1. Qual foi o valor total de vendas para cada modo de envio dos pedidos?
2. Quais mercados apresentaram o maior custo médio de envio?
3. A média do valor de venda atingiu a meta mensal de 350?
4. Qual categoria de produto apresentou o maior lucro médio?
5. Como a margem de lucro se comportou ao longo dos anos?

## Indicadores analisados

O projeto apresenta os seguintes indicadores:

- Valor total de vendas;
- Valor médio de venda;
- Custo médio de envio;
- Lucro médio;
- Margem média de lucro;
- Desempenho em relação à meta de vendas;
- Resultados por mercado, categoria, período e modo de envio.
